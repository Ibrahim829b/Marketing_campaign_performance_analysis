# Marketing_campaign_performance_analysis

## Project Overview
Analyzed Meta Ads campaign performance using Power BI to evaluate ad spend, CTR, conversion rates, ROI, and sales performance.



## Key Insights
- Identified top-performing campaigns.
- Optimized ad spend allocation.
- Contributed to an estimated 137% increase in sales.
